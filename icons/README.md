# Icon Files Required

This directory should contain the following PNG icon files:

- icon16.png (16x16)
- icon32.png (32x32)
- icon48.png (48x48)
- icon128.png (128x128)

You can create these icons using any image editor or online icon generator.

**Quick solution:** Use an online tool like:
- https://favicon.io/favicon-generator/
- https://www.icoconverter.com/

Just upload any image and select the required sizes.
